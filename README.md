# 🚀 Fakhri Portfolio - Digital Archive

Website portofolio modern yang terintegrasi dengan **Supabase** untuk menampilkan daftar tugas kuliah secara dinamis.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Custom Properties), JavaScript (ES6+)
- **Backend-as-a-Service:** Supabase (Database & Auth)
- **Typography:** Syne & Manrope (Google Fonts)

## 🔑 Konfigurasi Database
Proyek ini menggunakan tabel `assignments` pada Supabase. Pastikan Anda memiliki kolom berikut:
- `title` (text)
- `description` (text)
- `link` (text)
- `created_at` (timestamp)

## 📦 Cara Instalasi
1. Clone repositori.
2. Konfigurasi API Key Supabase Anda di `assets/js/index.js`.
3. Jalankan menggunakan Live Server di VS Code.
